


PDO is a Database Abstraction Layer.
It is "Universal," meaning it works with many database types
It is more secure and helps prevent SQL Injection.



 To create a connection, we need three arguments:

DSN: (The Address).

Username: (Database User).

Password: (User Password).






DSN stands for Data Source Name. It contains:

Driver:  mysql

Host: Server location host=localhost.

Database Name: dbname=blogdb

Charset: Language encoding  charset=utf8.



 Error Handling (Try / Catch)
English: We use try/catch blocks to handle connection errors gracefully.

Try: Executes the connection code.

Catch: Captures any Exception error if the connection fails.

$e->getMessage(): Displays the specific error message.




 By default, PDO is silent.
  We use setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION)
  to force PDO to throw exceptions when an error occurs.
